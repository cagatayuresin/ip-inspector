About

IP Inspector is a minimal, fast, single-file IP & CIDR analysis tool for DevOps, NetOps, and anyone working with networks. No backend, no tracking, just pure client-side analysis. Open source and ready for GitHub Pages.

Help

  • Enter an IPv4 address (e.g. 192.168.1.10) or a CIDR block (e.g. 10.0.0.0/24) in the input box and click Analyze or press Enter.
  • Results will show network, broadcast, mask, class, RFC scope, and more.
  • Use Export .txt to download a summary, or Copy Summary to copy to clipboard.
  • History is stored locally in your browser and can be cleared anytime.
  • For issues or contributions, visit the GitHub repository.
